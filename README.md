# Neural Network Robustness Project

## Objective
Analyze robustness and reliability of neural networks under noise and adversarial perturbations.

## Dataset
MNIST (via PyTorch)

## Models
- MLP (baseline)
- CNN (planned)

## Evaluation
- Accuracy
- Robustness under noise
- Adversarial attacks (FGSM)
- Calibration (ECE)
